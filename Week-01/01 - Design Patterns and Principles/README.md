Module 1 - Design Patterns and Principles

Overview:
This module introduces essential design principles and patterns (SOLID: SRP, OCP, LSP, ISP, DIP) and common design patterns (Creational, Structural, Behavioral). Focus is on theory plus practical exercises to write clean, maintainable, and scalable code.

Learning objectives:
- Understand SOLID principles and apply them.
- Implement Creational, Structural, and Behavioral patterns.
- Select appropriate patterns to solve design problems.

Hands-on: Complete the module exercises from the program GitHub repository.
